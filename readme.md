# Relify

**Relify** 是一个高性能、可扩展的统一消息路由核心。它旨在连接不同的即时通讯平台，支持复杂的路由拓扑和消息映射。

## ✨ 核心特性

*   **多架构模式**：支持 **Peer (对等网状)** 和 **Hub (中心辐射)** 两种架构模式。
*   **智能路由**：
    *   **Mirror (镜像)**：一对一房间映射，自动双向同步。
    *   **Aggregate (聚合)**：多对一映射，将多个群组消息汇聚到一个中心频道。
*   **自动建房 (Auto-Binding)**：无需手动配置复杂的房间 ID，核心根据流量自动建立绑定关系。
*   **全量消息映射**：
    *   支持消息 **编辑 (Edit)** 和 **撤回 (Delete)** 的跨平台同步。
    *   支持 **回复 (Reply)** 关系的 ID 自动翻译与降级处理。
*   **元数据同步**：自动同步发送者头像、昵称以及群组信息。
